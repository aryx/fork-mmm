MMM : astuces

Utilisation optimale du tachymètre

Si vous voulez surfer beaucoup avec un même MMM, faites comme suit: retailler la fenêtre initiale pour que seul le tachymètre reste visible (80x80 est la taille nominale), ou encore lancez mmm avec l'option -geometry 80x80+X+Y (avec X et Y donnant la position initiale). Ensuite, rendez cette fenêtre Sticky avec le window manager, afin que la fenêtre reste visible dans tous vos écrans virtuels. Vous pouvez maintenant ouvrir des fenêtres en cliquant avec le bouton 1 dans le tachymètre. Vous pouvez également sélectionner une URL avec la souris dans une fenêtre quelconque, puis cliquer avec le bouton 2 dans le tachymètre, ce qui aura pour effet d'ouvrir une nouvelle fenêtre sur cette URL.

Goodies

  • Le titre d'un document est un menu : il permet de voir les en-têtes de ce document. Certaines en-têtes ont une action (p.ex. Refresh) qui peut être activée par le menu.
  • Pour chercher un texte, utiliser Control-S. Cela fonctionne la fenêtre d'affichage HTML (mais incorrectement en présence de tables ou de frames), et dans la fenêtre d'affichage du source d'un document HTML.
  • La boîte de dialogue de sélection d'un nom de fichier (p.ex. pour sauver) comporte un mode électrique. Si la souris est dans une des listes, on peut taper des caractères pour se déplacer rapidement dans la liste.
  • Lorsqu'on sauve un fichier, si le premier caractère du nom de fichier est un pipe (|), le nom est utilisé comme une commande à laquelle le contenu du document courant est passé en entrée standard.
