MMM : tips

Optimal use of the tachymeter

If you usually do long Web browsing sessions with a single instance of the browser, then proceed as follows: Resize the initial window to that only the tachymeter remains visible (80x80 is the nominal size, or better, run mmm with the option -geometry 80x80+X+Y (where X and Y give the initial position). Then, make this window Sticky in your window manager, so that it remains visible in all your virtual screens. You can now open windows by clicking with mouse button 1 in the tachymeter. You can also select a URL in any window using the mouse, and the click with button 2 on the tachymeter to open a new MMM window on the document at this address.

Goodies

  • The document title is a menu : it displays the headers of the document. Some headers are active (e.g. Refresh).
  • To search for text, type Control-S. This works in HTML display windows (but incorrectly in the presence of tables and frames), and in the source windows as well.
  • The dialog box for selecting a file name (e.g. when saving) has an electric mode. If the mouse cursor is in one of the lists, you can type caracters for faster moving around the list.
  • When you save to a file, if the first character of the file name is a pipe (|), then the name is used as a command and executed with the document as standard input.
